# CIS 121 - Programming Fundamentals: Lab 3

This repository contains two C++ programs developed for CIS-121. These exercises demonstrate fundamental programming concepts, including mathematical computations, standard input/output, and data type casting.

## Repository Contents

### 1. Cylinder Area & Volume (`CylinderArea.cpp`)
* **Description:** A program that computes the volume and surface area of a cylinder based on user-provided radius and height in meters. 
* **Key Concepts:** Demonstrates the use of the `<cmath>` library, constant definitions for pi (`3.1416`), and the `pow()` function for exponentiation.

### 2. Stock Commission Calculator (`StockCommission.cpp`)
* **Description:** A financial calculator that determines the amount paid for stock alone, the commission amount, and the total amount paid based on a fixed price of $35.00 per share.
* **Key Concepts:** Highlights the use of the `<iomanip>` library for formatting floating-point numbers (`setprecision`, `fixed`, `showpoint`) and the use of `static_cast<int>` for converting final calculated doubles into integer output.
