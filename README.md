# CIS 121 - Programming Fundamentals: Lab 3

This repository contains C++ solutions for Lab 3 of the CIS 121 course. These programs focus on implementing mathematical formulas and financial calculations using standard C++ libraries.

---

## Included Exercises

### Cylinder Area & Volume (`CylinderArea.cpp`)
* **Purpose:** Calculates the surface area and volume of a cylinder based on user-defined radius and height.
* **Technical Implementation:** Utilizes the `<cmath>` library for power functions and defines a constant for $\pi$ ($3.1416$) to ensure calculation accuracy.

### Stock Commission Calculator (`StockCommission.cpp`)
* **Purpose:** Computes the cost of stock purchases, broker commissions, and the total transaction amount.
* **Technical Implementation:** Employs the `<iomanip>` library to format currency output and uses `static_cast<int>` to manage data type conversions for the final display.
